SignLingo — AI-Powered Sign Language Learning App

// Jakarta, Indonesia

SignLingo is an AI-powered web-based platform designed to make learning Indonesian Sign Language (BISINDO) more accessible and interactive. At its core is a sophisticated gesture recognition system capable of understanding 26 distinct complex gestures in real time.

The technical centerpiece is a Hybrid CNN LSTM spatiotemporal model — a deep learning architecture that combines the spatial feature extraction power of CNNs with the temporal sequence modelling of LSTMs. This enables the system to not just recognise static hand shapes, but to understand the motion and timing of gestures over time.

  • Optimised real-time inference latency to under 50 milliseconds, delivering seamless, instant visual feedback to users in a web-based platform.

A Deep Learning Framework for Early Forest Fire Detection

// Jakarta, Indonesia

This project tackles early-stage forest fire detection using state-of-the-art deep learning — with the goal of catching fires before they grow uncontrollable. The system detects smoke and early flame indicators, making detection possible well ahead of traditional sensors.

As Lead Developer, the architecture of choice was YOLOv8 — the latest YOLO evolution with anchor-free detection and decoupled prediction heads for cleaner predictions and lower false-positive rates compared to older architectures like YOLOv5.

A major focus was making the model deployable on resource-constrained edge devices like Raspberry Pi — meaning this system could realistically be embedded into forest monitoring stations without requiring cloud connectivity.

  • Developed a real-time computer vision framework using YOLOv8 to detect early-stage forest fires through smoke and flame on a custom-curated fire/non-fire dataset.
  • Achieved 93.2% mAP@0.5 and 91.3% precision — outperforming baseline YOLOv5 models through anchor-free detection and decoupled heads, significantly reducing false positives.
  • Optimised for edge deployment with ~43 FPS and 23ms per image inference, suitable for Raspberry Pi and other resource-constrained hardware in the field.
  • Leveraged YOLOv8's modern architecture to reduce false positive rates in challenging visual scenarios such as fog, mist, and cloud cover.

NLP-Powered Content Moderation Engine

// Jakarta, Indonesia

Online communities generate enormous volumes of text, and manually moderating toxic content at scale is simply not feasible. This project built an automated, real-time content moderation pipeline powered by modern transformer-based NLP — enabling platforms to detect and flag toxic behaviour before it causes harm.

The core model is DistilBERT — a distilled version of Google's BERT, designed to be significantly faster and lighter while retaining most of BERT's language understanding. Fine-tuned on the Jigsaw Toxic Comment Classification dataset, the model understands context, sarcasm, and subtle forms of harmful language beyond simple keyword matching.

The production-ready backend was built with Flask, exposing REST API endpoints that allow any application to submit text and receive a real-time toxicity classification response.
